# Gallery media

Public files in this folder are web-ready assets used by the site.

Photos:

- `photo-01.webp` through `photo-23.webp`

Videos:

- `video-01.mp4` through `video-11.mp4`
- `video-01-poster.webp` through `video-11-poster.webp`
- `hero-card-01.mp4` through `hero-card-04.mp4`
- `hero-card-01-poster.webp` through `hero-card-04-poster.webp`

Original camera/export files are kept under:

- `_source/photos/`
- `_source/videos/`

Use WebP for photos and H.264 MP4 for videos. Keep public filenames lowercase and predictable.
